## ChatGuard

ChatGuard is a secure end-to-end encrypted messaging app for platforms such as **Eitaa, Bale, and Soroush**, operating without servers or user accounts. All encryption keys are stored locally on the device, and messages are encoded as regular text to ensure privacy and avoid detection. Technical details are available in the [How it Works](https://amirroid.github.io/chatguard/how-it-works/) guide, with usage instructions provided in the [How to Use (English)](https://amirroid.github.io/chatguard/how-to-use/) and [How to Use (Persian)](https://amirroid.github.io/chatguard/fa/how-to-use/) documentation. The latest releases are available on [GitHub Releases](https://github.com/amirroid/ChatGuard/releases).
