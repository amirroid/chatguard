## ChatGuard

ChatGuard is a simple and secure app for end-to-end encrypted messaging on platforms like **Eitaa,
Bale, and Soroush**, working entirely without servers or accounts. All encryption keys are stored
locally on your device, and encrypted messages are disguised as normal text, keeping your chats
private and undetectable. Detailed information about how ChatGuard secures your messages can be
found in the [How it Works](https://amirroid.github.io/chatguard/how-it-works/) guide, and instructions for using the app are
available in the [How to Use (English)](https://amirroid.github.io/chatguard/how-to-use/)
or [How to Use (Persian)](https://amirroid.github.io/chatguard/fa/how-to-use/) guides. The latest versions of the app can be
downloaded [from GitHub Releases](https://github.com/amirroid/ChatGuard/releases).
