# ChatGuard

> Simple end-to-end encryption for Eitaa, Bale, Soroush, Telegram, and similar messaging platforms.

[Download Latest Release](https://github.com/amirroid/chatguard/releases){ .md-button .md-button--primary }
[GitHub Repository](https://github.com/amirroid/chatguard){ .md-button }

---

ChatGuard is an open-source application that provides end-to-end encrypted messaging without requiring servers, accounts, or centralized infrastructure.

All cryptographic operations are performed locally on the user's device, and encryption keys never leave the device. ChatGuard does not operate any servers and does not require user registration.

To remain compatible with existing messaging platforms, encrypted messages are encoded as ordinary-looking Persian text before being shared. This allows users to exchange protected messages through supported messengers while maintaining privacy and confidentiality.

## Features

* 🔐 End-to-end encryption
* 📱 Compatible with Eitaa, Bale, Soroush, Telegram, and similar platforms
* 🚫 No servers or cloud infrastructure
* 👤 No account registration required
* 🔑 Local key generation and storage
* 🔍 Open-source and transparent implementation
