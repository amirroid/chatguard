# چت‌گارد

> رمزنگاری سرتاسری ساده برای ایتا، بله، سروش، تلگرام و سایر پیام‌رسان‌های مشابه

[مخزن گیت‌هاب](https://github.com/amirroid/chatguard){ .md-button }

چت‌گارد یک نرم‌افزار متن‌باز است که امکان تبادل پیام‌های رمزنگاری‌شده را بدون نیاز به سرور، حساب کاربری یا هرگونه زیرساخت متمرکز فراهم می‌کند.

تمام عملیات رمزنگاری به‌صورت محلی روی دستگاه کاربر انجام می‌شود و کلیدهای رمزنگاری هرگز از دستگاه خارج نمی‌شوند. چت‌گارد هیچ سروری را مدیریت نمی‌کند و برای استفاده از آن نیازی به ثبت‌نام وجود ندارد.

برای حفظ سازگاری با پیام‌رسان‌های موجود، پیام‌های رمزنگاری‌شده به متنی شبیه متن عادی فارسی تبدیل می‌شوند و سپس از طریق پیام‌رسان‌های پشتیبانی‌شده ارسال می‌گردند. این رویکرد امکان تبادل پیام‌های محافظت‌شده را بدون نیاز به زیرساخت اضافی فراهم می‌کند.

## ویژگی‌ها

* 🔐 رمزنگاری سرتاسری
* 📱 سازگار با ایتا، بله، سروش، تلگرام و پیام‌رسان‌های مشابه
* 🚫 بدون نیاز به سرور یا زیرساخت ابری
* 👤 بدون نیاز به ثبت‌نام یا ایجاد حساب کاربری
* 🔑 تولید و نگهداری کلیدها به‌صورت محلی
* 🔍 متن‌باز و قابل بررسی
